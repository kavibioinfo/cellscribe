What CellScribe Does

End-to-end single-cell transcriptomics pipeline

📁

Multi-Format Input

Supports .h5ad, .h5 (10x), .mtx (Matrix Market), and .csv files

🔬

Automated QC

Quality control, normalization, and feature selection automatically

🤖

Deep Learning

scVI variational autoencoders for batch correction and latent representation

🎯

Cell Type Annotation

CellTypist integration predicts 42+ immune cell types with confidence scores

Differential Expression

Wilcoxon rank-sum test with volcano plots and adjusted p-values

Publication Plots

UMAP, violin plots, heatmaps, and interactive cell type charts
